3. 매개변수가 있는 함수를 만들 수 있나요? ㅇㅇ
- 매개변수?
  일종의 외부 데이터를 함수 내부로 전달하는 매개체 역할을 하는 변수.

  ```javascript
   function pr99(i){ //i가 매개변수!
    document.write(i +'x 2 ='+ (i*2),'<br />')
    document.write(i +'x 4 ='+ (i*4));
    }

  pr99(5); //데이터 // 매개변수==데이터 같게 된다.!
  ```
